# midefensaperu-backoffice

Panel admin de midefensaperu.com — verificación de abogados, resolución de disputas y estadísticas.

## Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS

## Features
- Verificación de abogados (documentos, colegiatura)
- Resolución de disputas
- Dashboard de estadísticas

## Dev
```bash
npm install
npm run dev
```
